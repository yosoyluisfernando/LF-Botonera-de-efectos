# Política de Seguridad / Security Policy

## Versiones Soportadas

Actualmente, solo la última versión estable (publicada en GitHub Releases) recibe actualizaciones de seguridad. Te recomendamos mantener tu aplicación siempre actualizada.

| Versión | Soportada |
| ------- | ------------------ |
| v1.1.x  | ✅ Sí              |
| < v1.1  | ❌ No              |

## Reportar una Vulnerabilidad

Por favor, **NO** abras un Issue público si descubres una vulnerabilidad de seguridad, ya que esto podría comprometer a todos los usuarios de la aplicación.

Si crees haber encontrado un problema crítico de seguridad en **LF Botonera de Efectos** (por ejemplo, ejecución remota de código, escalada de privilegios a través de los atajos globales, o vulnerabilidades en el servidor de actualización local), por favor repórtalo enviando un mensaje directo a los desarrolladores principales (o mediante los canales privados especificados en el perfil de GitHub del mantenedor).

Si el problema afecta a dependencias subyacentes como Tauri, Rodio, o Symphonia, evaluaremos la aplicación de un parche o actualización de la dependencia de inmediato.

Recibirás una respuesta en cuanto la vulnerabilidad haya sido analizada.

---

*For English speakers: Please **do not** open a public issue for security vulnerabilities. Send a private message or email to the repository owner instead.*
