# Política de Seguridad

Si encuentras una vulnerabilidad de seguridad en **LF Botonera de Efectos**, por favor **no abras un issue público**.

Usa primero el reporte privado de vulnerabilidades de GitHub, si está disponible en este repositorio. Si no aparece esa opción, contacta al mantenedor por un canal privado oficial indicado en el README o en el perfil de GitHub del proyecto.

La política completa está en [`Documentación/SECURITY.md`](Documentación/SECURITY.md).
